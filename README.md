A personal cybersecurity blog documenting my learning journey through ReDI School, TryHackMe, and beyond. I write about what I'm studying, what's clicking, and what's challenging me — from foundational concepts like the CIA Triad and networking to offensive security, certifications, and career paths.

Built with plain HTML, CSS, and JavaScript — no frameworks, no build tools, fully hosted on GitHub Pages.

[View the blog →](https://oriana11.github.io/Oriana-s-Blog)

## 🛠️ Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- [Fira Code](https://fonts.google.com/specimen/Fira+Code) + [Inter](https://fonts.google.com/specimen/Inter) + [Montserrat](https://fonts.google.com/specimen/Montserrat) via Google Fonts
- [Giscus](https://giscus.app/) for comments (GitHub Discussions)
- GitHub Pages for hosting
